To create movement

Transform property: it allows us to rotate, scale, move, skew, etc., elements.

  • transform: rotateY(angle), for defining a 3D rotation along the Y-axis.

Transform-style property: It specifies how nested elements are rendered in 3D space, it must be used together with the transform property.

  • transform-style: preserve-3d, for defining a perspective view for a 3D transformed element.

Transition

  • transition property: transform, for adding a transformation to the transition effect.
  • transition-duration: 2s, for defining the velocity of the transform (2s).

Backface-visibility property

  • backface-visibility: hidden, the backside is not visible.
